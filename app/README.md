# Country Info App

Приложение для Android, которое показывает информацию о странах мира, используя открытое REST API.

## 📱 О приложении

Приложение загружает список стран и отображает:
- Название страны
- Столицу
- Регион
- Население

Данные получены с [REST Countries API](https://restcountries.com/)

## 🛠 Технологии

- **Язык:** Kotlin
- **Сеть:** Retrofit2 + OkHttp
- **Асинхронность:** Coroutines
- **UI:** RecyclerView, XML

## 🚀 Запуск проекта

1. Клонировать репозиторий
2. Открыть в Android Studio
3. Синхронизировать Gradle
4. Запустить на эмуляторе или устройстве

## 📋 Требования

- Android minSdk 24
- Интернет для загрузки данных

## ⚠️ Примечание

Приложение использует параметр `fields` в запросе к API, чтобы получать только необходимые данные (ограничение API - не более 10 полей).
